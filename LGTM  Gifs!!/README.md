# What is this?

Retrieves a random LGTM GIF link from LGTM Gifs and copies it to the clipboard (with auto-paste).

Powered by LGTM Gifs: https://www.lgtmgifs.com

## Usage

Type `lgtmgifs` in Alfred, and a random GIF link in Markdown format will be copied and pasted automatically.

```
![LGTM](https://www.lgtmgifs.com/...)
```
